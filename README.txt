== Description ==

Connecting Element's wordpress integration for the Central Signup JSON Service

== Installation ==

This section describes how to install the plugin and get it working.

1. Upload the plugin folder 'ce-csjs' to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
